[umaar.github.io/meter-display/src/index.html](https://umaar.github.io/meter-display/src/index.html)

#### Or run this locally:

```sh
cd src
python -m SimpleHTTPServer
```

Then open up [http://localhost:8000/](http://localhost:8000/)

#### To run tests

```sh
npm i
make test
```
