
# Meter Display

## Requirements

- Node (tested on `6.2.0`)

### Get Started

```
node -v # v6.2.0
npm i
```

### Build

```
npm run build
```

Now open up `dist/index.html` in your browser

### Development

```
npm start
```

### Tests (incomplete)

```
npm install -g testem

# Unit tests
npm test

# Experimental headless browser tests
npm run test-qunit
```
